# mucru.az — Claude Project Context

## Layihə haqqında

**mucru.az** — Azərbaycanca hədiyyə səbəti e-ticarət platformasıdır.

### Məhsul strategiyası
- **Əsas məhsul (Phase 1):** Duxu (ətir) — gül səbətlərinin içinə qoyulan əsas hədiyyə
- **Phase 2:** Gül (çiçək) əlavəsi — qızılgül, lalə və digər çiçəklər
- **Gələcək genişləmə:** Çikolad, oyuncaq, dekorativ elementlər

### Hədəf bazar
- Azərbaycan (əsas)
- Azərbaycan dili — birincil dil
- Bakı şəhəri — ilk çatdırılma zonası

### Biznes modeli
- Online sifariş + çatdırılma
- Hədiyyə səbəti kurasiyası (duxu + gül kombinasiyaları)
- Xüsusi günlər: ad günü, toy, Novruz, 8 Mart, Sevgililer günü
